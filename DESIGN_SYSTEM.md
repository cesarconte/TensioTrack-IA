# Design System — TensioTrack

## 🎨 Paleta de Colores
- **Primario**: Indigo-600 (`#4f46e5`) - Acciones principales.
- **Salud/Corazón**: Rose-500 (`#f43f5e`) - Pulso y alertas.
- **Normal**: Emerald-500 (`#10b981`) - Valores óptimos.
- **Alerta**: Amber-500 (`#f59e0b`) - Valores elevados.
- **Fondo**: Slate-50 (`#f8fafc`) - Superficie base.

## 🔡 Tipografía
- **Sans**: Inter (UI general, legibilidad alta).
- **Display**: Outfit (Encabezados, toque moderno).
- **Mono**: JetBrains Mono (Datos numéricos, precisión).

## 📏 Espaciado y Layout
- **Grid**: Sistema de 12 columnas para desktop.
- **Padding**: Escala de 4px (base 1rem/16px).
- **Bordes**: Radio de 12px (`rounded-xl`) para tarjetas y botones.

## 📱 Breakpoints
- `sm`: 640px (Mobile-First base)
- `md`: 768px (Tablets)
- `lg`: 1024px (Laptops)
- `xl`: 1280px (Desktop)

## 🧩 Componentes Base (shadcn/ui)
- Button, Card, Input, Dialog, Tooltip, Toast, Progress.
